Emerald offline wallet command line interface.

Usage:
  emerald server [options] [--port <port>] [--host <host>]
  emerald account list [options] [--show-hidden]
  emerald account [un]hide [options] [<uuid>|<address>]
  emerald account new [options] [--security-level <level>] [--raw] [<name> [<description>]] [<passphrase-file>|-]
  emerald account strip [options] (<uuid>|<address>) [<passphrase-file>|-]
  emerald account update [options] (<uuid>|<address>) [<name> [<description>]]
  emerald account import [options] [<keyfile>|-]
  emerald account export [options] [<uuid>|<address>] [<keyfile>|-]
  emerald transaction sign [options] [--raw] <from> <nonce> <to> <gas> <gas-price> (<value>|<data>) [<passphrase-file>|-]
  emerald -V | --version
  emerald -h | --help
  emerald [-v | -vv | -vvv]

Options:
  -h, --help                                Show this message
  -V, --version                             Show current version
  -v, --verbose                             Use verbose output (-vvv very verbose output)
  -q, --quiet                               No output printed to stdout
      --host <host>                         Listen host [default: 127.0.0.1]
      --port <port>                         Listen port [default: 1920]
      --base-path <path>                    Base directory path, if omitted default os-specific value will be used:
                                              + Mac OS X: ~/Library/Emerald
                                              + Linux: ~/.emerald
                                              + Windows: %USERDIR%\.emerald
      --show-hidden                         Show also hidden accounts
      --raw                                 Expect raw private key instead of a random one or extracted by given passphrase
      --security-level <normal|high|ultra>    Level of security for cryptographic operations [default: normal]
  -c, --chain <mainnet|testnet>               Chain name [default: mainnet]
      --chain-id <id>                       Chain id number [default: 61]
